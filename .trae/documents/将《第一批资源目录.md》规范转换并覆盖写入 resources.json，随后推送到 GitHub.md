* 从`/f:/学科资源整理/第一批资源目录.md`解析所有资源条目，统一映射为站点使用的JSON结构。

* 覆盖写入到`/f:/学科资源整理/data/resources.json`，保证页面（如`resources/fea-theory.html`）正常渲染。

* 提交并推送到GitHub远端仓库。

## 数据映射规则

* 字段对齐：

  * 标题→`title`

  * 链接→`url`

  * 类型（讲义/视频/课程/PDF/开源程序/工具）→`type`（doc|video|code|tool）

  * 语言（zh/en）→`language`

  * 标签（以中文分隔符拆分）→`tags`（数组）

  * 简介→`description`

  * 难度（入门/进阶/高级/研究生/实操）→`difficulty`（整数1-5；用于星级渲染）

  * 最近校验→`updatedAt`（ISO日期`YYYY-MM-DD`，取当天日期）

  * `id`：按链接域名与唯一标识生成（如bilibili用`bili-<BV号>`，MIT OCW用`mit-<课程号>`，官方站点用`comdyn-<slug>`），避免中文转拼音引入不稳定性。

* 分类映射：

  * 线性代数→`category: math`，`subCategory: linear-algebra`

  * 微积分→`category: math`，`subCategory: calculus`

  * 概率与统计→`category: math`，`subCategory: probability-statistics`

  * 数值分析→`category: math`，`subCategory: numerical-methods`

  * 优化→`category: math`，`subCategory: optimization`

  * PDE与有限元理论→`category: fea-theory`，`subCategory: pde`或`finite-element-method`（按条目主题自动区分）

  * 连续体力学→`category: math`，`subCategory: continuum-mechanics`

  * 中文精选（线代/教程）→按内容归入`math`或`fea-theory`对应子类（不保留“中文精选”作为独立分类）

  * 有限元与计算力学（中文）：

    * 中文课程/讲义→`category: fea-theory`，`subCategory: finite-element-method`

    * 开源程序→`category: programming`，`subCategory: finite-element-implementation`

    * 软件教程→`category: software`，`subCategory: abaqus`

    * 前置课程（弹性力学）→`category: math`，`subCategory: continuum-mechanics`

* 难度映射：

  * 入门→1；入门-进阶→2；进阶→3；高级→4；研究生级/高级（标注“高级/研究生级/⭐⭐⭐⭐⭐”）→5；实操→3（若含基础入门则为2）。

## 实施步骤

1. 读取并解析`第一批资源目录.md`：按章节`##`与条目块解析“标题/链接/平台/类型/语言/难度/标签/简介/最近校验”等字段。
2. 应用分类与难度映射，生成规范对象数组；按站点现有`resources.json`字段顺序输出。
3. 覆盖写入`/data/resources.json`（完整替换）。
4. 快速校验：

   * 语法检查：确保JSON可解析且数组非空。

   * UI契合：`difficulty`为整数1-5，避免`fea-theory.html`星级渲染出现空星。

   * 采样检查：随机抽取5项比对`title/url/tags/category`与原MD一致。

## 推送到GitHub
